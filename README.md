
## Description

CRM API is an API for building CRM's. Get an API REST boilerplate to kickstart the backend of your project. 

## how to use it: 
- yarn install
- yarn start:dev
- Define your business logic
- Get a REST API with a MongoDB database to access your data.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
